# LiveSync — Real-Time Video Conferencing Platform

LiveSync is a full-stack real-time video conferencing platform built with React, TypeScript, Node.js, Socket.IO, and WebRTC.

## Current Progress

- Project initialized
- React + TypeScript frontend setup
- Node.js + Express backend setup
- Socket.IO signaling server setup
- Room-based real-time chat test added

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Socket.IO Client

### Backend
- Node.js
- Express.js
- Socket.IO
- TypeScript

## Core Concepts

- HTTP for normal API communication
- Socket.IO for real-time signaling
- WebRTC for peer-to-peer video/audio streaming
- Rooms for isolated meetings
